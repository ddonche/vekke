Vekke Board

By Intl. Vekke Council ~ 5 min read

Vekke is played on a 6x6 board. There are a couple variations. Both styles are fully interchangeable and equivalent in rules; choice is aesthetic/preference-based (or Order-themed for intersection).

Grid Board

The default board is the standard grid board, which is closer to checkerboard style games (chess, checkers, etc). Unlike the checkerboard pattern, all Vekke squares are the same color. Some variations may come with the grid coordinates written either on each square or around the outer edge of the board.

And below you can see an example of the grid style board with unit tokens on it.

grid board with tokens

Intersection Board

Another board style you may see is the intersection board, which is closer to Go style games (Go, Pente, etc.). This board uses dots at intersections instead of grid spaces, and is a little more difficult to read than the grid board. However, it has lines from each dot indicating legal move directions, which the grid board does not have. Colors of the dots in the game also use the primary color of the Order you are in; the one below matches Order of the Turtle.

Tip
Notice how the corner dots each have some diagonals missing. This is because you cannot go in those directions; the intersection board is helpful for learning directions.

the intersection style board

And below you can see an example of the intersection style board with unit tokens on it.

intersection board with tokens

Switching Board Styles

You can switch board styles at any time during a game by pressing the B key on your keyboard.

Grid Notation

Grid space coordinates are named by the column and row in which they appear. Columns go from a to f (left to right as read from B brake side of the board) and rows go from 1 to 6 (bottom to top as read from B brake side of the board).

Note
The grid style board has the coordinates listed on each space, while the intersection style does not.
Important
Grid coordinates use lower case letters whenever we refer to them. This is because capital letters mean something else in Vekke notation. The board may sometimes have capital letters for readability.

Considerations

Because Vekke's board is smaller than chess (8x8) and Go (19x19 grid), it is a much tighter battlespace. This is because the routes, which dictate all movement, only allow movement up to a maximum of 4 spaces in a given direction. This differs from games like Chess, which has pieces that can travel the entire length of the board (Queen, Rook, Bishop). However, to compensate for this, Vekke features a mechanic called flanking, which is where a piece that reaches the edge of the board with distance remaining continues movement from the other side of the board.

Style inspiration Chess/Checkers Go/Pente
Placement points Squares Intersections (dots)
Direction indicators None (pure coordinates) Lines show legal directions
Readability Easier for beginners Slightly harder, but teaches directions
Coordinates shown Yes (on spaces or edges) No
